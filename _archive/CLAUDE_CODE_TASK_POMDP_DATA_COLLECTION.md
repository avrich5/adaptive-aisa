# Задание: сбор данных для проекта Adaptive AI Strategy Advisor (POMDP)

## Контекст

Запускается исследовательский проект: переработка AI Strategy Advisor в адаптивную систему на базе POMDP-формализма. Три оси состояний:

- **User State** — скрытое состояние пользователя (explorer, position manager, yield seeker, professional и т.д.)
- **Market State** — режим рынка (rising, falling, sideways, high/low volatility, transitional)
- **Strategy State** — профиль каждой стратегии в разных режимах

Перед стартом разработки нужно собрать фактическую картину: что уже есть в коде и данных, чего не хватает, что придётся строить с нуля.

**Задача Claude Code: не строить, а инвентаризовать.** Никаких изменений в существующем коде. Только чтение и документирование.

Репозиторий: `ai-trading-strategy-advisor`, путь `backend/app/orchestrator`.
